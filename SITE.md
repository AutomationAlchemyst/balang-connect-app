# Site Vision: Balang Connect - Premium Coastal Catering

## 1. Vision & Goals
A high-end catering platform for "Balang" drinks, flavored with a "Breezy Balang" coastal aesthetic. The goal is to provide a seamless booking experience for corporate and private events while fostering a sense of community through Infaq and shared stories.

## 2. Project Configuration
- **Stitch Project ID:** 6162856176582044019
- **Design Theme:** Coastal Glassmorphism (See DESIGN.md)

## 3. Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS + Custom Glassmorphism Utilities
- **Backend:** Firebase (Real-time DB & Auth)
- **Deployment:** App Hosting / Vercel
